Step-by-Step Guide — Build a Working Voice Agent in Under 1 Hour

1. What You're Building

An AI phone agent that:

  • Answers your phone number 24/7 with natural voice
  • Searches your knowledge base (Obsidian, documents) during live calls
  • Triggers actions — outbound calls, SMS, reminders
  • Costs under $15/month (vs $99+ for prefab tools)
Prerequisites: You need a computer that can run Node.js and stay online. If it sleeps, the agent sleeps. For 24/7 uptime, run on a $5/mo VPS or Raspberry Pi.

2. Architecture Overview

YOU (calls Twilio number) ↓ Twilio (routes call) ↓ ElevenLabs Conversational AI │ ← Natural voice ↓ Webhook → Your Bridge Server │ ← POST with transcription ├─→ /search (query Obsidian) ├─→ /call (outbound via Twilio) └─→ /sms (send message) ↓ Return result → ElevenLabs → spoken response

Three services, one bridge, one goal.

3. Step 1: ElevenLabs Setup

Create an Agent

  1. Go to elevenlabs.io → Sign up or log in
  2. Navigate to Conversational AI → Agents
  3. Click Create Agent → Name it (e.g., "Maya")
  4. Pick a voice: Sarah (warm, natural) or Bella (bright, professional)
  5. System prompt: "You are my personal assistant. Answer calls warmly, take messages, and help schedule appointments."
VIP Tip: Keep "first message" under 15 seconds. Example: "Hi, this is Maya. How can I help you today?"

Get Your Agent ID

Find your Agent ID (looks like agent_xxxxxxxxxxxxxxxxxxxxxxxx) in the dashboard URL or settings. Save it.

4. Step 2: Twilio Phone Number

  1. Go to twilio.com → Sign up → Complete "quest" for $15 free credit
  2. Buy a phone number (Phone Numbers → Manage → Buy a number)
  3. Pick one with Voice + SMS capability (~$1.15/mo)

Configure the Webhook

You'll wire this after your bridge is running. For now, note down:

  • Account SID (starts with AC_)
  • Auth Token (secret — never share)
  • Phone Number
Wait on webhook config: Don't set the webhook URL yet. Your bridge needs to be live first.

5. Step 3: Build the Bridge Server

Create the Project

mkdir my-voice-agent cd my-voice-agent npm init -y npm install express axios dotenv

Server Code (server.js)

require('dotenv').config(); const express = require('express'); const axios = require('axios'); const app = express(); app.use(express.json()); // Health check app.get('/health', (req, res) => { res.json({ status: 'ok', agent: 'voice-agent' }); }); // Main webhook app.post('/maya/webhook', async (req, res) => { console.log('Webhook received:', req.body); const { tool_name, parameters } = req.body; try { if (tool_name === 'obsidian_search') { const results = await searchObsidian(parameters.query); return res.json({ success: true, result: results }); } if (tool_name === 'outbound_call') { await makeCall(parameters.phone, parameters.message); return res.json({ success: true, result: 'Calling now' }); } return res.json({ success: true, result: 'No action needed' }); } catch (err) { console.error(err); return res.status(500).json({ success: false, error: err.message }); } }); async function searchObsidian(query) { const resp = await axios.get('http://127.0.0.1:27123/search/', { params: { query }, headers: { 'Authorization': 'Bearer ' + process.env.OBSIDIAN_TOKEN } }); return resp.data.slice(0, 3).map(r => r.filename + ': ' + r.excerpt).join('\\n'); } async function makeCall(to, message) { const twilio = require('twilio')(process.env.TWILIO_SID, process.env.TWILIO_TOKEN); await twilio.calls.create({ to, from: process.env.TWILIO_PHONE, url: process.env.BRIDGE_URL + '/twiml?msg=' + encodeURIComponent(message) }); } app.listen(process.env.PORT || 8765, () => { console.log(`Bridge running on port ${process.env.PORT || 8765}`); });

Environment (.env)

PORT=8765 TWILIO_SID=AC_xxxxxxxx TWILIO_TOKEN=your_auth_token TWILIO_PHONE=+1XXXXXXXXXX BRIDGE_URL=https://your-tunnel-url.trycloudflare.com OBSIDIAN_TOKEN=your_obsidian_token

6. Step 4: Cloudflare Tunnel (Public URL)

Your computer is behind a router. You need a public URL for Twilio/ElevenLabs to reach it.

  1. Install cloudflared:
    # macOS brew install cloudflared # Ubuntu sudo apt install cloudflared
  2. Run the tunnel:
    cloudflared tunnel --url http://localhost:8765
    Copy the URL (e.g., https://abc-123.trycloudflare.com)
  3. Make it permanent: Sign up for Cloudflare Zero Trust (free), create a named tunnel with a fixed subdomain.
Heads up: Free tunnel URLs change on restart. For production, use a named tunnel or run as background service: nohup cloudflared tunnel --url http://localhost:8765 &

7. Step 5: Wire ElevenLabs → Your Bridge

  1. ElevenLabs → Conversational AI → your agent → Settings
  2. Paste webhook URL: https://your-tunnel-url.trycloudflare.com/maya/webhook
  3. Add these External Tools:
    Tool NameMethodURL
    obsidian_searchPOST/maya/webhook
    outbound_callPOST/maya/webhook
CRITICAL — ElevenLabs Tool Schema Trick: Here's the exact JSON that passes validation:
{ "name": "obsidian_search", "description": "Search my notes", "parameters": { "type": "object", "properties": { "query": { "type": "string", "description": "What to search for" } }, "required": ["query"] } }
If the dashboard still rejects this, create via API instead.

8. Step 6: Wire Twilio → ElevenLabs

  1. Twilio Console → Phone Numbers → Manage → Active numbers
  2. Click your number → Voice & Fax
  3. A call comes in → Webhook
    https://api.elevenlabs.io/v1/convai/twilio/outbound?agent_id=YOUR_AGENT_ID // Method: POST

Now when someone calls your Twilio number, it routes to ElevenLabs, which talks to your bridge.

9. Step 7: Obsidian Knowledge Base

  1. Obsidian → Settings → Community Plugins
  2. Install "Local REST API" → Enable
  3. Note port (default 27123) and generate API token
  4. Add token to .env
Privacy: Local REST API only works on your machine. Notes never leave your computer unless the bridge explicitly sends data.

10. Step 8: Test the Full Stack

  1. Start bridge: node server.js
  2. Start tunnel: cloudflared tunnel --url http://localhost:8765
  3. Call your Twilio number from your cell
  4. Say: "Search my notes for projects I'm working on"
  5. Check bridge console for the webhook hit
If nothing happens: Check Twilio logs. Common issues: wrong agent ID, cloudflared not running, missing /maya/webhook path.

11. Step 9: Outbound Calling

To make your agent call YOU back:

  1. Add the outbound_call tool to your agent
  2. Tell agent: "When asked, call me back with the outbound_call tool."
