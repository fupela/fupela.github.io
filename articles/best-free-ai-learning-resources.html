AI learning ranking

Best free AI learning resources, ranked.

If you are trying to learn AI, do not start by buying random courses. Start with official docs, free beginner courses, hands-on notebooks, and daily research/news sources. This is the Jedaiflow starter ranking.

Updated May 12, 2026 · Free and low-friction resources first

How to use this list

Pick one resource from the level you are actually at. Beginners should start with Elements of AI or Microsoft AI concepts. Builders should add OpenAI/Anthropic docs and DeepLearning.AI. Coders can move into Google ML Crash Course, Kaggle, Harvard CS50 AI, and fast.ai.

The Jedaiflow AI learning path

  1. Week 1: Elements of AI + Microsoft AI concepts.
  2. Week 2: OpenAI and Anthropic prompting docs. Build a daily prompt notebook.
  3. Week 3: DeepLearning.AI prompt engineering + one small workflow.
  4. Week 4: Kaggle or Google ML Crash Course. Learn enough ML vocabulary to understand what models can and cannot do.
  5. Ongoing: Scan official AI news and Hugging Face papers. Save only the resources that change what you build.
