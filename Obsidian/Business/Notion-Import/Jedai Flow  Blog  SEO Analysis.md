## Critical SEO Issues to Fix
[ ] No Google Analytics — install tracking so you can measure traffic
[ ] No sitemap.xml — Google can’t efficiently discover and index your pages
[ ] No structured data (JSON-LD) — missing Article schema markup for rich results
[ ] No og:image on posts — no preview image when shared on social media
[ ] No Twitter card meta tags
[ ] Social links in footer point to "#" — need to link to real profiles
