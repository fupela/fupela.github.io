The 5-Minute Voice Agent Checklist

Print this. Check each box. Build your AI voice agent today.
1

ElevenLabs Account + Agent

Tools: elevenlabs.io (free tier) & phone
  • Sign up at elevenlabs.io
  • Create a Conversational AI agent (not just a voice)
  • Pick a voice (Sarah is warm and natural)
  • Note your Agent ID for later
2

Twilio Phone Number

Tools: twilio.com/quest ($15 free credit)
  • Sign up and get a phone number (<$5/mo)
  • In Console, configure Voice & SIP → Webhook URL → incoming
  • Point webhook to ElevenLabs (they provide the URL)
  • Test with a call — you should hear your agent's voice
3

Bridge Server (your computer)

Tools: Node.js, cloudflared tunnel
  • Create a Node.js server with one POST /webhook endpoint
  • Install cloudflared → run `cloudflared tunnel --url http://localhost:PORT`
  • Copy the public URL (ends in `.trycloudflare.com`)
  • Save it as a fixed route in Zero Trust for permanence (optional)
  • In ElevenLabs dashboard, paste the public URL as webhook
4

Custom Tools (the magic)

Tools: ElevenLabs dashboard
  • In dashboard → Tools → Add External Tool
  • Name: obsidian_search — Method: GET/POST — URL: your bridge + /search
  • Name: outbound_call — Method: POST — URL: your bridge + /call
  • Tip: If ElevenLabs rejects your schema, use the exact JSON from the guide — API validation is finicky
5

Knowledge Base (Obsidian / Notes)

Tools: Obsidian + Local REST API plugin
  • Enable Obsidian Local REST API plugin (community plugins)
  • Note the port (default 27123)
  • In your bridge server, query http://localhost:27123/search/ with the user's question
  • Return top results to ElevenLabs as agent response
6

Test End-to-End

Tools: your phone + the bot
  • Call your Twilio number
  • Say: "What projects am I working on?" (should trigger Obsidian search)
  • Say: "Call me tomorrow at 9am" (outbound_call tool)
  • Check your bridge logs for the webhook payload
